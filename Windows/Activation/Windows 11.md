## Remove product key, upgrade to Windows 11 Pro, change to actual product key.

- slmgr.vbs
  - /dli
    - Display license information
  - /ipk
    - Install product key
  - /ato
    - Activate windows
  - /upk
    - Uninstall product key
  - /cpky
    - Clear produce key from the registry
